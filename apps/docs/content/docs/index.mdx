This template helps you launch a **shadcn registry with documentation** in minutes.

It combines a registry-ready project structure with a preconfigured docs app, so you can focus on building and publishing components instead of setting up boilerplate.

## What You Get

- A starter setup for shipping components through a shadcn registry
- A docs site to document usage, examples, and installation
- A clean foundation you can extend with your own design system and content

## Next Steps

- Update this docs content to match your registry and components
- Add or edit component pages under `content/docs/components`
- Start publishing your registry items and keep docs in sync
